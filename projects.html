BioPrint Interactive Anatomy

Collaborated with a team of four developers to design and build BioPrint Interactive Anatomy, an AR-based mobile application for medical education, aimed at enhancing anatomy learning through immersive 3D interaction. We developed a functional prototype in Unity (C#), integrating the AR technology to allow users to explore a detailed, interactive 3D heart model with dynamic UI controls. Implemented core features, including a main menu navigation system and real-time user interaction with anatomical structures. The project progressed through initial prototyping phases before discontinuation.
